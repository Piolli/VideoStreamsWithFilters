### Installing:
```
1. Just copy dylibs by dragging files from folder to XCode project root files hierarchy.
2. Copy dylibs to /Xcode/DerivedData/Lab3_VideoStream...*..../Build/Products/Debug/Lab3_VideoStream.app
3. Run application.
```
